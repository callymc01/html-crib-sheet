Media Elements

This page summarises the media-related HTML elements I've learned so far.

Image Element

The <img> element embeds images into a webpage. It is a void element (no closing tag).

Essential Attributes

  • src — image file path or URL
  • alt — text description for accessibility
  • title — optional tooltip text
Show example 

<img src="cat.jpg" alt="A cute cat" title="Cat photo">

Audio Element

The <audio> element embeds audio files such as music or sound effects.

You can include multiple <source> elements for better browser compatibility.

Useful Attributes

  • src — audio file path or URL
  • controls — shows play/pause UI
  • autoplay — starts automatically
  • loop — repeats when finished
  • muted — starts muted
Show example 

<audio controls>
    <source src="sound.mp3" type="audio/mpeg">
    <source src="sound.ogg" type="audio/ogg">
</audio>

Video Element

The <video> element embeds video content.

Multiple <source> elements can be used for compatibility. Autoplay usually requires muted for modern browsers.

Useful Attributes

  • src — video file path or URL
  • controls — shows play/pause UI
  • autoplay — starts automatically
  • loop — repeats when finished
  • muted — starts muted
  • poster — image shown before playback
  • source — used inside <video> to list file types (mp4, webm, ogg)
Show example 

<video controls poster="preview.jpg">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
</video>

Iframe Element

The <iframe> element embeds external content such as YouTube videos or maps.

It is a replaced element — its content comes from an external resource.

Important Attributes & Values

  • allow — permissions for the iframe (e.g., autoplay, clipboard-write)
  • accelerometer — allows motion sensor access
  • autoplay — allows auto-playing media
  • clipboard-write — allows writing to clipboard
  • encrypted-media — allows DRM-protected content
  • gyroscope — allows orientation sensor access
  • web-share — allows native share dialogs
  • referrerpolicy — controls what referrer info is sent
  • strict-origin-when-cross-origin — common referrer policy value
  • allowfullscreen — allows fullscreen mode
Show example 

<iframe 
    src="https://www.youtube.com/embed/example"
    allow="autoplay; encrypted-media; fullscreen"
    allowfullscreen>
</iframe>
